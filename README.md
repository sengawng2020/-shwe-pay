## Getting Started

### View the Landing Page

Simply open `index.html` in your web browser, or serve it with any web server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve

# Using PHP
php -S localhost:8080
```

Then navigate to `http://localhost:8080` in your browser.
